# 🚀 Space GK Quiz

A full-stack web-based quiz application built using **Python Flask** with a futuristic animated space background.

🌍 **Live Website:**  
👉 https://space-gk-quiz.onrender.com

---

## ✨ Features

- 🔐 User Login & Registration
- 🧠 Multiple Choice GK Questions
- 📊 Progress Bar Tracking
- 🎨 Animated Milky Way Background
- 👨‍💼 Admin Dashboard
- 🗄️ SQLite Database
- ☁️ Deployed on Render

---

## 🛠 Tech Stack

- Backend: Python (Flask)
- Frontend: HTML, CSS, JavaScript
- Database: SQLite
- Deployment: Render
- Server: Gunicorn

## 🔧 How To Run Locally

```bash
git clone https://github.com/Shantanu0015/space-gk-quiz.git
cd space-gk-quiz
pip install -r requirements.txt
python app.py
```

Then open:

```
http://127.0.0.1:5000
```
